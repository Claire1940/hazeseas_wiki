<Section icon={Map} title="Haze Seas シービーストの出現場所と概要">

**Haze Seasのシービースト**（「レッド・シービースト（Red Sea Beast）」とも呼ばれる）は、ゲーム中盤から終盤における最も手強い挑戦の1つです。レベル700のスーパーボスに分類されるこの巨大な水棲爬虫類は、立ち回りと防御タイミングをしっかりと理解することを要求します。事前の準備がないプレイヤーにとってはその圧倒的な攻撃力に圧倒されがちですが、攻撃モーションが非常に分かりやすいため、安定して周回（ファーミング）するのに最適なターゲットとなっています。

ボスを見つけるには、マップの端へと向かう必要があります。ボスはクルーフラッグ（Crew Flag）のすぐ近くにあるロッキー・ピラーズ（Rocky Pillars）地域に出現します。このエリアは高低差のある地形とオープンウォーター（開けた海）が特徴であるため、戦闘を開始する前にスタミナ管理と足場の位置取りをしっかりと行うことが極めて重要です。

| ボスの属性 | プロファイル詳細 |
| :--- | :--- |
| **ボス名** | シービースト (レッド・シービースト / Sea Beast) |
| **推奨レベル** | レベル700 (スーパーボス) |
| **出現地域** | ロッキー・ピラーズ (Rocky Pillars) |
| **最寄りの目印** | クルーフラッグ / スカイウォークトレーナーの柱 |
| **主な危険要素** | 高い瞬間火力とスタン（気絶）ギミック |
| **推奨戦闘スタイル** | 高火力の遠距離ビルド、または連携の取れたチームプレイ |

<Callout type="info" title="出現時の警告">
シービーストは非常に高いダメージを与えてくるため、一瞬で体力を削られる可能性があります。レベル700向けとなっていますが、高防御ビルドや連携のとれたチームがない状態で、近接での接近戦を挑むことは基本的におすすめしません。
</Callout>

このクリーチャーに挑戦しようと考えているプレイヤーは、事前にビルドを準備しておく必要があります。氷（Ice）、砂（Sand）、糸（String）、バリア（Barrier）などの高火力で遠距離から攻撃できる悪魔の実が非常に有効です。さらに、周囲の柱を移動し、戦闘中に海に落下するのを防ぐために、スカイウォーク（Sky Walk）などの移動スキルの解放は実質的に必須となります。

</Section>

<Section icon={Swords} title="攻撃パターンと戦闘戦略">

**Haze Seasのシービースト**戦を生き抜くには、その厳格な3つの攻撃サイクルを暗記する必要があります。ランダムに技を選択する他の多くのボスとは異なり、このスーパーボスは固定された連続ループでスキルを実行します。このシーケンスさえ理解してしまえば、いつ回避し、いつ防御アビリティを有効にし、いつ最大火力のスキルを叩き込むべきかを正確に予測できるようになります。

ボスは「アクアティック・ファイアボール（Aquatic Fireball）」、「ロアー（Roar）」、「シービーム（Sea Beam）」の3つの異なる技を使用します。これらは常にこの正確な順序で発動するため、戦闘のリズムは常に一定に保たれます。

| 攻撃名 | ダメージ値 | 判定の大きさ | 技の特性とギミック効果 |
| :--- | :--- | :--- | :--- |
| **アクアティック・ファイアボール (Aquatic Fireball)** | 2500ダメージ | 極めて大きい | 予備動作の長い、巨大な青い飛び道具 |
| **ロアー (Roar)** | 0ダメージ | 中 | 見聞色の覇気を完全に無効化する範囲スタン |
| **シービーム (Sea Beam)** | 1500ダメージ | 小 | 発生が非常に早い、弾速の速い長射程レーザー |

<Callout type="warning" title="覇気の妨害">
ロアー（咆哮）攻撃自体は直接的なダメージを与えませんが、見聞色の覇気を無効化されるため非常に危険です。素早く反応しなければ、次に繰り出されるシービームをまともに喰らうことになります。
</Callout>

ダメージを受けずにこのサイクルを乗り切るには、戦闘中に以下の戦術的な回避シーケンスを実践してください：

<Steps>
  <Step num="1" title="アクアティック・ファイアボールを回避する">
    ボスの予備動作が始まったら、横方向にダッシュする準備をしましょう。青い火球の判定は非常に大きいため、通常の移動だけでは避けきれないことがほとんどです。タイミングよく回避やテレポートを使用して、爆発範囲から離脱してください。
  </Step>
  <Step num="2" title="ロアーを受ける、または回避する">
    至近距離にいる場合、ロアーを簡単に避けることはできません。短いスタンを受け入れるしかありませんが、すぐに次の防御行動を取れるよう準備してください。ユーティリティスキルを再発動できるように指を構えておきましょう。
  </Step>
  <Step num="3" title="見聞色の覇気を再有効化する">
    ロアーのスタンが解けた瞬間に、手動で見聞色の覇気を再起動します。これを行うことで、直後に飛んでくる高速のシービームをキャラクターが自動で回避できるようになります。
  </Step>
  <Step num="4" title="遠距離反撃を叩き込む">
    シービームを回避し、ボスが短い硬直状態に入ったら、最大火力の遠距離スキルを叩き込みましょう。その窓が閉じたら、サイクルが再びアクアティック・ファイアボールから始まるため、ステップ1を繰り返す準備をします。
  </Step>
</Steps>

</Section>

<Section icon={Sparkles} title="スカイウォークの柱ハメ技ストラテジー">

**Haze Seasのシービースト**をソロで、かつ何度も敗北するリスクを冒さずに周回したいプレイヤーのために、非常に信頼性の高い「ハメ技（安全地帯攻略）」が存在します。この方法は、ロッキー・ピラーズの垂直な地形を利用して、ボスの経路探索と攻撃ベクトルを無効化します。ボスの物理攻撃が届かない位置に身を置くことで、混沌としたスーパーボス戦をシンプルな「的当てゲーム」へと変えることができます。

この戦略は、スカイウォークトレーナー（Sky Walk Trainer）NPCがいる特定の柱を中心に行われます。この柱は高さがあり、水面からも距離があるため、理想的な安全地帯（セーフゾーン）として機能します。

<CardGrid cols={3}>
  <Card title="ソロハメ技法">
    - **立ち位置**: スカイウォークトレーナーの柱の上に直接立ちます。
    - **戦闘スタイル**: 遠距離から悪魔の実のアビリティや銃攻撃を連発します。
    - **安全度**: 極めて高い。物理攻撃はこの高さまで届きません。
  </Card>
  <Card title="チーム戦闘法">
    - **立ち位置**: 周囲の複数の柱に分散して配置します。
    - **戦闘スタイル**: ボスのターゲット（ヘイト）を交互に受けてダメージを分散させます。
    - **安全度**: 普通。積極的な回復と連携が必要です。
  </Card>
  <Card title="標準近接戦闘法">
    - **立ち位置**: 柱の根元近くの、海面と同じ高さで戦います。
    - **戦闘スタイル**: 高リスクな剣や格闘スタイルでの戦闘。
    - **安全度**: 低い。高火力の範囲（AOE）攻撃を非常に受けやすいです。
  </Card>
</CardGrid>

<Callout type="tip" title="おすすめの悪魔の実">
柱ハメ技を使用する場合、高低差に対応したロックオンを持つ実や、長距離を直進する飛び道具を持つ実（氷や砂など）が非常に優れた性能を発揮します。視界を遮られることなく、安全な場所からボスに安定してダメージを与え続けることができます。
</Callout>

ハメ技を実行するには、スカイウォークを使用してトレーナーの柱の頂上まで登ります。この有利な位置から、下にいるボスをロックオンします。遠距離攻撃を絶え間なく放ちましょう。ボスがアクアティック・ファイアボールやシービームを発動しそうになったら、柱の端から少し後ろに下がり、柱の縁を自然の遮蔽物として利用します。これにより、飛んでくる飛び道具を完全に遮断でき、体力をほとんど減らさずにボスを安全に周回できます。

</Section>
