<Section icon={Map} title="種族の稀有度と選択システム">

Haze Seasの世界において、種族はキャラクタービルドの土台となります。ゲーム開始時、プレイヤーにはランダムに種族が割り当てられますが、これは「種族スピン（race spins）」を使用することで変更可能です。これらのスピンは、ゲーム内ショップ、デイリーログイン報酬、またはキャンディを再ロールと交換できるハロウィンイベントのような季節イベントに参加することで入手できます。限られたリソースを無駄にすることなくビルドを最適化するためには、稀有度（レアリティ）のティアを理解することが極めて重要です。

<Section icon={Swords} title="V1種族のステータスと戦闘でのメリット">

Haze Seasのすべての種族は、獲得した時点で即座にパッシブバフを提供します。これらのV1ボーナスは、主に移動速度、体力（HP）、スタミナ回復に焦点を当てています。新規プレイヤーにとって、**Haze Seasの種族**システムはシンプルに見えるかもしれませんが、これらのわずかなパーセンテージが序盤のボス戦の勝敗を分けることがよくあります。

人間（Human）種族は最も一般的であるにもかかわらず、驚くほど高い体力ブーストを提供するため、レジェンダリー層をアンロックしていないプレイヤーにとって実用的な「タンク」の選択肢となります。逆に、魚人（Fishborn）のような特化型種族は、海に頻繁に行くプレイヤーにとって不可欠であり、悪魔の実の能力者が持つ本来の弱点を緩和してくれます。

| 種族名 | 歩行速度 | 体力/スタミナ | 独自のV1パッシブ |
| :--- | :--- | :--- | :--- |
| **人間 (Human)** | +25% | +70% HP | なし |
| **獣人 (Beastborn)** | +20% | +5% スタミナ | 基礎移動速度の上昇 |
| **空島人 (Skyborn)** | 標準 | 標準 | 最初のスカイウォーク（月歩）のブースト+50% |
| **魚人 (Fishborn)** | 標準 | 標準 | 泳ぎ速度+50% |
| **悪魔 (Demon)** | 標準 | 標準 | 攻撃命中時に2.5%のライフスチール（HP吸収） |
| **竜人 (Dragon Born)**| 標準 | +10% HP / 回復力 | 最初のスカイウォーク（月歩）のブースト+25% |
| **Dの一族 (DClan)** | 標準 | +5% HP / スタミナ | 覇王色の覇気（消費スタミナ250） |

<Callout type="warning" title="スタミナ管理">
Dの一族（DClan）のユーザーは、V1での「覇王色の覇気」の使用に注意する必要があります。250のスタミナ消費は序盤のレベルでは非常に大きく、タイミングを誤ると回避やアビリティの使用ができなくなる可能性があります。
</Callout>

</Section>

<Section icon={Target} title="V2覚醒への道のり">

V1からV2への移行は、Haze Seasにおいてプレイヤーが達成できる最大のパワー急上昇（パワースパイク）です。このプロセスは単なる購入ではなく、戦闘スキルと忍耐力が試される過酷な一連の試練を伴います。レベルキャップの要件に達するまではこの旅を始めることができないため、経験豊富なプレイヤーのみがこれらの進化した形態にアクセスできるようになっています。

V2クエストラインはすべての種族で共通です。つまり、人間であろうとDの一族のメンバーであろうと、同じ試練を通じて己の価値を証明しなければなりません。

<Steps>
  <Step num="1" title="レベル要件の達成">
    V2クエストNPCが相手をしてくれるようになるには、**レベル2200**に達している必要があります。それより前に開始しようとしても、NPCはあなたの要求を無視します。
  </Step>
  <Step num="2" title="海獣（Sea Beast）狩り">
    最初の試練では、**海獣（Sea Beast）**を見つけて倒す必要があります。これらは外海に出現し、安全に倒すにはある程度の船の耐久度か、射程の長い悪魔の実のアビリティが必要です。
  </Step>
  <Step num="3" title="宝探し（スカベンジャーハント）">
    マップ上の様々な島に散らばっている**宝箱を50個**集めます。これは、スターターアイランド（Starter Island）や海軍基地（Marine Base）などの密度の高いエリアを訪れることで効率よく完了できます。
  </Step>
  <Step num="4" title="ボス「ゼニス」との対決">
    高レベルボスである**ゼニス（Zenith Boss）**を撃破します。この戦闘をクリアするとクエストが完了し、ユニークな刀剣がドロップするチャンスもあります。
  </Step>
  <Step num="5" title="最終支払い">
    戦闘の試練をすべて完了した後、覚醒を完了するためにNPCに**1,000,000ベリー**と**100ジェム**を支払う必要があります。
  </Step>
</Steps>

<Callout type="success" title="クエストの継続性">
1つの種族でV2をアンロックした後、種族を切り替えた場合でもクエストを再試行できます。ただし、新しく覚醒させたい種族ごとに、ベリーとジェムの費用を再度支払う必要があります。
</Callout>

</Section>

<Section icon={Sparkles} title="V2進化：強化された能力">

種族をV2に覚醒させることで、パッシブボーナスがゲームの勝敗を左右する強力なメカニクスへと変化します。例えば、魚人（Fishborn）種族は、悪魔の実の能力者に完全な水無効化を付与するため、間違いなく最も強力なユーティリティ種族となります。これにより、戦闘中に海に落ちるという最大の脅威が排除されます。

悪魔（Demon）や空島人（Skyborn）などの他の種族は、移動中や戦闘中にトリガーされるアクティブスタイルのパッシブを獲得し、戦闘にさらなる戦略の深みをもたらします。

| 種族 (V2) | 強化ステータス | 新しい能力/メカニクス |
| :--- | :--- | :--- |
| **人間 (Human)** | +50% 速度 / 70% HP | 見聞色の回避+1 |
| **獣人 (Beastborn)** | +50% 速度 / 10% スタミナ | 機動力が大幅に向上 |
| **空島人 (Skyborn)** | スカイウォークの向上 | **グライド（滑空）**: スペースキー長押しで落下速度が2倍遅くなる |
| **魚人 (Fishborn)** | 泳ぎ＆歩行速度2倍 | 悪魔の実の能力者向けの**水無効化** |
| **悪魔 (Demon)** | 3% ライフスチール | **地獄の業火（Hellfire）**: ダッシュ時にダメージエリアを生成 |
| **竜人 (Dragon Born)**| +10% HP / グライド | 落下速度が2倍遅くなる |
| **Dの一族 (DClan)** | +10% HP ＆ スタミナ | **覇王色**: 消費スタミナが50に減少 |

<Callout type="info" title="専門家の意見">
竜人（Dragon Born）V2にはグライド能力が追加されますが、一部のベテランプレイヤーは、特定のボスタンク（盾役）シナリオにおいてV1の素の回復スケーリングを好む場合があります。しかし、コンテンツの99%においては、V2へのアップグレードの方が優れています。
</Callout>

</Section>
