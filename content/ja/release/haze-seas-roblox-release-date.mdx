<Section icon={Shield} title="悪魔の実のメカニクスと一覧">

悪魔の実は、Haze Seasにおける主な力の源です。現在**34種類の実**が存在し、レア度（コモン、アンコモン、レア、レジェンダリー、ミシカル）に分類されています。フルーツディーラーにRobuxを支払いたくないプレイヤーにとって、出現タイマーを把握することは極めて重要です。

**悪魔の実の出現メカニクス:**
- **出現タイマー**: 新しい実が**60分**ごとにマップ上に出現します。
- **消滅タイマー**: 出現から**20分**以内に回収されない場合、実は消滅します。
- **VIPサーバー**: VIPサーバーでは、最も古い実が置き換わる前の**上限が5個**に制限されています。

| 実の名前 | レア度 | 主な用途 | ダメージタイプ |
| :--- | :--- | :--- | :--- |
| **Dough（モチモチ）** | ミシカル | PvPコンボ / ボス戦 | 超人系（パラミシア） |
| **Dragon（ウオウオ）** | ミシカル | 変身 / 範囲攻撃（AoE） | 動物系（ゾオン） |
| **Magma（マグマグ）** | レア | 海上イベント / 周回 | 自然系（ロギア） |
| **Light（ピカピカ）** | レア | 移動 / 周回 | 自然系（ロギア） |
| **Ice（ヒエヒエ）** | レア | クラウドコントロール（CC） | 自然系（ロギア） |
| **Chop（バラバラ）** | コモン | 初心者向けの防御 | 超人系（パラミシア） |

<Callout type="info" title="自然系（ロギア）の強み">
初心者プレイヤーには、**自然系（ロギア）の実**（モクモク、メラメラ、ピカピカなど）を狙うことを強くお勧めします。これらの実は「元素反射（Elemental Reflex）」を提供し、自分よりレベルの低いNPCからの物理攻撃を無効化するため、周回効率が大幅に向上します。
</Callout>

<Section icon={Sparkles} title="種族のレア度と育成ボーナス">

Haze Seasにおける種族は、パッシブバフや見た目の特徴を決定します。ほとんどのプレイヤーは人間（Human）としてスタートしますが、種族スピンを使用することで、**Dの一族（D. Clan）**や**ドラゴノイド（Dragonoid）**などの強力なバリアントを解放できます。各種族には特定の排出率が設定されており、非常に入手が困難なものもあります。

**種族ティアと効果:**
- **ミシカル (0.1%)**: **Dの一族（D. Clan）** - 体力自動回復、移動速度、追加の月歩（SkyWalk）、覇気（Haki）ボーナスを提供する究極の種族。
- **レジェンダリー (1%)**: **ドラゴノイド**および**ルナリアン** - 耐久性と垂直方向の機動力に特化。
- **レア (8.9%)**: **鬼（Demon）**（速度）および**天竜人（Celestial Dragon）**（お金）。
- **アンコモン (35%)**: **ミンク**、**魚人**、**サイボーグ**、**空島人（Skypeian）**。
