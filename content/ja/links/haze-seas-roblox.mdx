<Section icon={Sparkles} title="悪魔の実と出現メカニズム">

悪魔の実は、Haze Seas Roblox における強力なビルドの要です。現在34種類のユニークな実が用意されており、プレイヤーは自然系（ロギア）、超人系（パラミシア）、動物系（ゾオン）の中から選択する必要があります。各タイプは、レベリング、ボス狩り、PvP戦闘など、異なるプレイスタイルに応じた利点を提供します。

実の経済は特定のタイマーによって管理されており、プレイヤーは高レアリティの実を見つける確率を最大化するためにこれらを追跡する必要があります。

| メカニズム | 値 | プレイヤー戦略 |
| :--- | :--- | :--- |
| **出現タイマー** | 60分 | 1時間ごとに主要な島をチェックする |
| **消失タイマー** | 20分 | 出現が告知されたら迅速に移動する |
| **VIP制限** | 5個 | プライベートサーバーでの収集と安全な保管に最適 |
| **在庫更新** | 4時間 | Discordの在庫チャンネルで神話級アイテムを確認する |

<Section icon={Trophy} title="2026年 戦闘ティアリスト">

6月のリワーク以降、Haze Seas Roblox のメタは安定してきました。個人の好みも重要ですが、特定の性能を持つ実や武器は高レベルコンテンツにおいて明らかに他を圧倒しています。このティアリストでは、終盤の進行に最適な選択肢を分類しています。

| カテゴリ | Sティア (メタ) | Aティア (強力) | Bティア (実用的) |
| :--- | :--- | :--- | :--- |
| **悪魔の実** | モチ、ウオ（龍）、トリ（不死鳥） | ネコ（レオパルド）、ドク、マグ | ピカ、ジキ、ヒエ |
| **剣** | 夜（Yoru）、閻魔、三刀流 | 黄金の杖、雷雨、秋水 | ビスento V2、オペオペの刃 |
| **格闘スタイル** | 竜爪拳、サイボーグ | 魚人空手、エレクトロ | 黒足、コンバット |
| **種族** | Dの一族、ドラゴノイド | ルナリア族、悪魔 | ミンク族、魚人、サイボーグ |

<Callout type="info" title="神話級のレアリティ">
**モチ（Dough）** と **ウオ（Dragon）** は、その高いダメージスケーリングと変身能力により、依然として最も切望される実です。ただし、そのポテンシャルを完全に引き出すには、膨大なマスタリーポイントが必要です。
</Callout>

### 種族のレアリティとリロール率
種族はビルドを決定づけるパッシブバフを提供します。神話級のティアは非常に稀であるため、種族スピンは慎重に使用してください。

| 種族 | レアリティ | 排出率 | 主なボーナス |
| :--- | :--- | :--- | :--- |
| **Dの一族** | 神話級 | 0.1% | 覇王色の覇気+1、速度、回復 |
| **ドラゴノイド** | 伝説級 | 1.0% | 体力回復、ジャンプ力、覇気 |
| **ルナリア族** | 伝説級 | 1.0% | 空中歩行追加、スタミナ、火球 |
| **悪魔** | レア | 8.9% | 歩行速度+50%、体力、スタミナ |
| **セレスティアル** | レア | 8.9% | クエスト報酬+15%、バランス型 |
