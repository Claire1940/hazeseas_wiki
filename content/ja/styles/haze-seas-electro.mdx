<Section icon={Swords} title="戦闘メカニクスとスキル比較">

**haze seas electro**のスキルセットは、「電撃と畏怖」というコンセプトを中心に設計されています。素早い打撃と放電を組み合わせることで、プレイヤーはターゲットを足止めしながら、絶え間なくダメージを与え続けることができます。

リワークで利用可能な他の格闘スタイルと比較すると、エレクトロは黒脚（Black Leg）の手軽さと、竜（Dragonic）の圧倒的なパワーの中間に位置しています。

| 格闘スタイル | ティア | 主なメリット | 最適な使用例 |
| :--- | :--- | :--- | :--- |
| **Dragonic（竜）** | **S** | 圧倒的なダメージ | 終盤のPvP / ボス戦 |
| **Electro（エレクトロ）** | **A** | **速度 ＆ スタン** | **高速周回 / PvPコンボ** |
| **Fish-Man Karate（魚人空手）** | **A** | バランスの取れた射程 | 海上イベント / PvE |
| **Black Leg（黒脚）** | **B** | 序盤のアップグレード | 中盤の進行 |
| **Combat（格闘）** | **C** | なし | 初期レベル |

**エレクトロの主なスキル特徴:**
- **エレクトリック・ダッシュ (Electric Dash)**: 進路上にいる敵にダメージを与えながら高速移動するスキル。
- **サンダー・クラップ (Thunder Clap)**: 周囲のすべてのターゲットに2秒間のスタンを付与する局所的な範囲攻撃（AoE）。
- **ライトニング・バラージ (Lightning Barrage)**: 電撃をまとったパンチの連打。多段ヒットダメージを与え、短時間、敵の体力回復を阻害する。

<Callout type="success" title="プロのコンボのコツ">
**サンダー・クラップ**のスタンを当てた直後に、**閻魔（Enma）**の斬撃のような高火力の剣スキルや、神話級の悪魔の実の能力を叩き込みましょう。スタンによって、最も強力な技を確実に命中させることができます。
</Callout>

<Section icon={Shield} title="エレクトロ・ビルドの最適化">

**haze seas electro**を真にマスターするには、スタイル単体だけでなく、キャラクタービルド全体を考慮する必要があります。エレクトロはスピードに依存しているため、種族やサブ武器の選択が第2の海（Second Sea）での強さを左右します。

**エレクトロユーザーにおすすめの種族:**
- **ミンク族 / Mink (アンコモン)**: エレクトロにとっての王道。歩行速度とジャンプ力のボーナスがエレクトロの機動力と重複するため、敵から攻撃を食らうことがほぼ不可能になります。
- **Dの一族 / D. Clan (神話級)**: 莫大なスタミナと体力自動回復を提供し、エネルギー切れを起こすことなくエレクトロのスキルを連発できます。
- **サイボーグ / Cyborg (アンコモン)**: 耐久力と体力自動回復を底上げするため、「タンク寄り」の近接プレイスタイルを好むプレイヤーに有用です。

**武器のシナジー:**
エレクトロは格闘スタイルであるため、近接スロットを占有します。そのため、エレクトロの弱点である「長距離戦」を補う武器と組み合わせる必要があります。

| 武器 | 属性/レア度 | シナジーの理由 |
| :--- | :--- | :--- |
| **黄金のスタッフ (Golden Staff)** | レジェンダリー剣 | 雷に関連するテーマ。空島を起源とする共通点。 |
| **夜 / 黒刀 (Yoru / Dark Blade)** | 神話級の剣 | エレクトロでスタンさせたターゲットを仕留めるための高火力。 |
| **海獣のハンマー (Sea Beast Hammer)** | レジェンダリー武器 | 強力なスタン蓄積による完全なクラウドコントロール。 |

<Callout type="info" title="ステータス配分">
エレクトロ中心のビルドでは、**近接（Melee/Strength）**と**防御（Defense）**を優先してください。移動用に自然系（ロギア）の実を使用している場合、ハイブリッドビルドでない限り、**実（Fruit）**のステータスは優先度を下げてください。
</Callout>
