<Section icon={Sparkles} title="Frutas do Diabo e Mecânicas de Spawn">

As Frutas do Diabo são a base de qualquer build poderosa no Haze Seas Roblox. Com 34 frutas exclusivas disponíveis atualmente, os jogadores devem escolher entre os tipos Logia (Elemental), Paramecia (Natural) e Zoan (Animal). Cada tipo oferece vantagens distintas para diferentes estilos de jogo, como grinding, caça a chefes ou combate PvP.

A economia das frutas é regida por cronômetros específicos que os jogadores devem acompanhar para maximizar suas chances de encontrar spawns de alta raridade.

| Mecânica | Valor | Estratégia do Jogador |
| :--- | :--- | :--- |
| **Timer de Spawn** | 60 Minutos | Verifique as ilhas principais exatamente a cada hora |
| **Timer de Despawn** | 20 Minutos | Mova-se rapidamente assim que um spawn for anunciado |
| **Limite VIP** | 5 Frutas | Melhor para farm privado e armazenamento seguro |
| **Atualização de Estoque** | 4 Horas | Verifique o canal de estoque no Discord para itens míticos |
